# Raven Chat

An anonymous chat application built with Go and HTMX.

## Features

- Anonymous messaging
- Real-time chat using WebSockets
- Lightweight frontend with HTMX

## Tech Stack

- **Backend**: Go
- **Frontend**: HTMX
- **Communication**: WebSockets

## Getting Started

```bash
go run main.go
```
